Portfolio Risk Analysis

Python code for Value at Risk (VaR) calculation:


import numpy as np
import pandas as pd
from scipy import stats

# Calculate 95% Value at Risk
def calculate_var(returns, confidence_level=0.95):
    """Calculate VaR using historical simulation method"""
    return np.percentile(returns, (1 - confidence_level) * 100)

# Portfolio analysis for Q4 2025
portfolio_returns = np.array([-0.02, 0.03, 0.01, 0.04, -0.01])
var_95 = calculate_var(portfolio_returns, 0.95)

print(f"95% VaR: ${var_95 * portfolio_value:,.2f}")
# Output: 95% VaR: $1,234,567.89

Data Infrastructure

SQL query for revenue aggregation across divisions:


SELECT 
    division_name,
    SUM(revenue) as total_revenue,
    SUM(net_income) as total_profit,
    ROUND(SUM(net_income) / SUM(revenue) * 100, 2) as profit_margin,
    COUNT(DISTINCT customer_id) as unique_customers
FROM 
    financial_transactions
WHERE 
    transaction_date BETWEEN '2025-10-01' AND '2025-12-31'
    AND status = 'completed'
GROUP BY 
    division_name
HAVING 
    SUM(revenue) > 1000000
ORDER BY 
    total_revenue DESC;
Our data warehouse processes over 50M transactions daily. This query runs in under 2 seconds thanks to our optimized infrastructure. Real-time analytics enable faster decision-making.
